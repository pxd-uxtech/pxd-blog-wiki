# pxd 블로그 개념 위키

LLM이 pxd 블로그 글을 읽고 정리한 개념 페이지 모음.

## UX 이론
- [게슈탈트 법칙과 시각적 통일성](concepts/게슈탈트-법칙과-시각적-통일성.md) — 통일성(Unity)은 디자이너 요소들 사이에 시각적·형태적 유사성이 존재해 보는 이가 조화, 일치, 일관성을 느끼는 것
- [기호학과 인덱스](concepts/기호학과-인덱스.md) — 기호의 세 유형(아이콘·인덱스·심벌) 중 대상체와 인과적 연결을 갖는 인덱스(Index)의 개념과 UI 적용
- [디자인 주도 혁신](concepts/디자인-주도-혁신.md) — 로베르토 베르간티의 Design-Driven Innovation: 사용자 관찰 대신 의미(meaning) 재정의를 통한 혁신 접근
- [ISO UI 표준](concepts/ISO-UI-표준.md) — ISO 9241·13407·9241-210으로 이어지는 사용성과 UX의 국제 표준 계보
- [밀러의 매직 넘버](concepts/밀러의-매직-넘버.md) — 1956년 조지 밀러가 발표한, 인간의 단기 기억 용량은 7±2개라는 법칙
- [모델 휴먼 프로세서](concepts/모델-휴먼-프로세서.md) — 인간의 지각·인지·운동 시스템으로 구성된 1983년 Card·Moran·Newell의 인지 모델
- [사용자 중심 디자인](concepts/사용자-중심-디자인.md) — UCD(사용자 요구 반영)와 HCD(인간 인지·신체 특성 반영)의 차이, 제프 래스킨과 도널드 노먼의 관점
- [어포던스](concepts/어포던스.md) — 디자이너 사이에서 가장 자주 쓰이는 전문 용어 중 하나이지만, 정작 정의가 혼란스러운 개념
- [진화심리학과 디자인](concepts/진화심리학과-디자인.md) — 대칭성·평균성·아기 스키마 등 진화심리학으로 설명하는 아름다움과 귀여움의 원리
- [포카요케](concepts/포카요케.md) — 도요타 품질 기법을 UX에 적용한 오류 방지 설계 원칙
- [피츠의 법칙](concepts/피츠의-법칙.md) — 1954년 폴 피츠가 발표한 운동 행동 모델로, 목표 영역을 클릭하는 데 걸리는 시간 예측
- [행동경제학과 UX](concepts/행동경제학과-UX.md) — 전망 이론·손실 회피·앵커링 등 행동경제학 개념의 UX 설계 적용
- [행동심리학과 강화이론](concepts/행동심리학과-강화이론.md) — 스키너의 조작적 조건화와 강화 스케줄, UX 게이미피케이션과 중독성 패턴의 이론적 배경
- [힉의 법칙](concepts/힉의-법칙.md) — 선택지 수가 늘어날수록 결정 시간이 증가한다는 법칙
- [미디어 방정식과 CASA](concepts/미디어-방정식과-CASA.md) — 사람은 미디어를 실제 사람처럼 대한다 — 무의식적 사회적 반응의 원리
- [문화심리학과 UX](concepts/문화심리학과-UX.md) — 리처드 니스벳의 동서양 사고 차이와 UX 설계 함의
- [컬처코드와 무의식적 문화 코드](concepts/컬처코드와-무의식적-문화-코드.md) — 클로테르 라파이유의 각인 발견 방법론과 문화 코드 5원칙

- [버스 정보 안내판 정보 설계](concepts/버스-정보-안내판-정보-설계.md) — 서울 BIT 번호 정렬 규칙과 정류장 맥락을 고려한 버스 노선도 리디자인 사례
- [사용자 여정 분석](concepts/사용자-여정-분석.md) — 사용자가 목적 달성을 위해 거치는 단계와 경험을 파악하는 방법론

- [아이트래킹](concepts/아이트래킹.md) — 시선 추적 기술로 사용자의 시각적 주의 패턴을 분석하는 UX 리서치 방법론
