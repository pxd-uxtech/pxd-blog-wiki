# pxd 블로그 개념 위키

LLM이 pxd 블로그 글을 읽고 정리한 개념 페이지 모음.

## 개념 목록

- [AI 코딩 도구 활용](concepts/AI-코딩-도구-활용.md) — Cursor AI 등 AI 코딩 도구의 실무 활용과 코파일럿 비교
- [BX 디자인과 브랜드 자산 운용](concepts/BX-디자인과-브랜드-자산-운용.md) — 인하우스 BX 디자이너의 브랜드 자산 운용과 인지 전략
- [CSS 로딩 애니메이션](concepts/CSS-로딩-애니메이션.md) — CSS/JS 기반 로딩 애니메이션 구현과 대기 경험 설계
- [UI 컴포넌트 용어](concepts/UI-컴포넌트-용어.md) — 드롭다운, GNB, 슬라이더 등 표준 UI 컴포넌트 용어 정리
- [UX 교육과 대학원](concepts/UX-교육과-대학원.md) — UX 전문가 양성을 위한 교육 과정과 대학원 선택 기준
- [UX와 UI의 차이](concepts/UX와-UI의-차이.md) — UI는 보편적 인간 대상의 분석, UX는 주관적 인간 대상의 공감
- [게슈탈트 법칙과 시각적 통일성](concepts/게슈탈트-법칙과-시각적-통일성.md) — 7가지 게슈탈트 법칙을 통한 시각적 통일성 구현
- [디자인 시스템](concepts/디자인-시스템.md) — 디자인 원칙, UI 패턴, 컴포넌트, 코드를 포괄하는 종합 세트
- [순서도와 프로그래밍 사고](concepts/순서도와-프로그래밍-사고.md) — 순서도를 통한 논리적 사고 훈련과 코드 변환
- [웹 아키텍처](concepts/웹-아키텍처.md) — 웹서버와 WAS의 역할 분리와 효율적 운영 구조
- [정보 구조 설계 IA](concepts/정보-구조-설계-IA.md) — 정보 구조 설계의 원칙과 사용자 중심 분류 체계
- [정보 디자인 원칙](concepts/정보-디자인-원칙.md) — 데이터를 의미 있게 전달하는 시각화 원칙
- [텍스트 임베딩과 데이터 시각화](concepts/텍스트-임베딩과-데이터-시각화.md) — 텍스트 임베딩 기반 데이터 시각화 방법론
- [피츠의 법칙](concepts/피츠의-법칙.md) — 클릭 거리·크기와 사용성의 정량적 관계
- [한글 입력 설계](concepts/한글-입력-설계.md) — 자소 빈도 분석 기반의 한글 자판 설계 원칙
