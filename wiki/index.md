# pxd 블로그 개념 위키

LLM이 pxd 블로그 글을 읽고 정리한 개념 페이지 모음.

## 개념 목록

- [UX와 UI의 차이](concepts/UX와-UI의-차이.md) — UI는 보편적 인간 대상의 분석, UX는 주관적 인간 대상의 공감
- [디자인 시스템](concepts/디자인-시스템.md) — 디자인 원칙, UI 패턴, 컴포넌트, 코드를 포괄하는 종합 세트
- [순서도와 프로그래밍 사고](concepts/순서도와-프로그래밍-사고.md) — 순서도를 통한 논리적 사고 훈련과 코드 변환
- [웹 아키텍처](concepts/웹-아키텍처.md) — 웹서버와 WAS의 역할 분리와 효율적 운영 구조
- [한글 입력 설계](concepts/한글-입력-설계.md) — 자소 빈도 분석 기반의 한글 자판 설계 원칙
