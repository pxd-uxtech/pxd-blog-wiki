# 사용자 여정 분석

사용자 여정 분석은 사용자가 특정 목적을 달성하기 위해 거치는 단계와 경험을 체계적으로 파악하는 방법론이다. 서비스의 사용 목적이 명확한 경우(예: 택시 호출, 모바일 뱅킹), 핵심 여정의 각 단계를 분해하고 경쟁 서비스와 비교하면 UX 개선 기회를 효과적으로 발견할 수 있다.

택시 호출 앱 비교 분석(카카오 T, UT, Uber, 타다)에서는 동일한 목적[호출 → 배차 → 승차 → 하차]을 갖는 서비스들도 세부 여정에서 차이를 보임을 확인했다. 예를 들어 출발지·도착지 입력 순서, 자동 위치 설정 여부, 배차 후 안내 방식 등이 서비스마다 달랐고, 이러한 차이가 사용자 경험의 质감을 결정한다. 분석에서는 서비스 차량 수나 배차 속도 같은 외부 조건은 제외하고 UX 요소만 비교함으로써 순수한 경험 차이에 집중했다.

카카오 뱅크 UX 사례에서는 상품 안내·약관 동의·계좌 개설 여정 전체를 사용자 관점에서 재설계했다. 기존 인터넷 뱅킹 앱이 '전문 용어와 복잡한 설명으로 상품 이해 장벽을 높이는' 패턴을 갖고 있는 반면, 카카오 뱅크는 슬로건과 3가지 요약 정보로 상품 특징을 즉시 파악 가능하게 했다. 이처럼 사용자 여정 분석은 경쟁사와의 비교를 통해 자사의 개선 방향을 도출하는 데 효과적이다.

## 핵심 내용
- 사용자 여정 = 목적 달성을 위해 거치는 단계의 흐름
- 동일 목적 서비스 간 여정 비교로 UX 차이점과 개선 기회 도출 가능
- 외부 조건(배차 속도 등)과 UX 요소를 구분해서 분석해야 순수한 경험 비교 가능
- 여정의 각 단계에서 필수 정보 수집·제공 설계가 핵심
- 사용자가 여정 중 맞닥뜨리는 혼란·장벽 지점이 개선 우선순위
