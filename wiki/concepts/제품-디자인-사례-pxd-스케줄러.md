# 제품 디자인 사례 — pxd 스케줄러

**pxd 스케줄러**는 2005년 한 직원이 개인 시간 관리를 위해 만든 종이 시간표에서 출발해, 회사 규모와 프로젝트 수가 늘어나면서 전 직원이 함께 쓰는 시간 관리 도구로 자연스럽게 확산된 사내 제품이다. 웹 프로그램 등 여러 시도를 거쳤지만 결국 하루를 '계획(왼쪽)'과 '실행 내용(오른쪽)'으로 나눠 적는 종이 스케줄러 형식으로 정착했고, 이후 한 UX 행사 후원 요청을 계기로 외부에 대량 배포할 제품으로 재디자인하게 되었다.

재디자인은 **pxd 아이덴티티 컬러 적용, 사용 안내 예시 페이지 삽입, 위클리 스케줄러와 연간 달력의 공존, 탁상 거치가 가능한 형태, 'UX 디자인 회사가 만드는 스케줄러란 무엇인가'라는 관점 반영**이라는 다섯 가지 목표를 세우고 진행됐다. 시간 테이블은 직원들의 실제 사용 행태를 관찰해 다시 그렸고, 필기 글씨 크기를 고려한 줄 간격을 여러 차례 테스트로 결정했으며, 내용이 없을 때도 지면이 허전해 보이지 않도록 그래픽 요소와 정보량의 균형을 조절했다. 연간 달력은 별도 제본이나 페이지마다 삽입하는 방식 대신 접히는 형태로 처리해 완성도와 부피 사이의 균형을 맞췄다.

형태 설계에서 가장 공을 들인 부분은 **탁상형 구조**였다. 기존 공책형 스케줄러는 필기에는 안정적이었지만 책상 공간을 많이 차지했고, 기존 탁상형은 E자 모양으로 꺾이는 연결 구조 때문에 눕혀서 쓸 수 없다는 한계가 있었다. pxd는 내부 연결 방식과 종이의 텐션을 이용해 눕혀서도 쓸 수 있으면서 기존 E자 구조를 벗어난 구조를 고안했고(특허출원), 인쇄소와 여러 차례 회의·샘플 제작을 거쳐 대량 생산 가능성을 검증했다. 겉표지에는 p·x·d 세 글자를 상징하는 세 개의 타공을 내 제품 특징을 아이콘으로 드러냈는데, 타공 시 재생지 특유의 누런 단면이 보이는 문제를 발견하고 표지 전체를 흰색 종이로 다시 제작해 균일한 화이트 톤을 유지했다. 겹판 구조인 뒷받침대의 내부 연결 종이에는 내구성과 슬라이딩감을 위해 코팅 처리를 더했다.

이 제작기는 완성된 결과물을 소개하는 데 그치지 않고, 사내 도구가 외부용 상품으로 재설계되는 과정에서 **목표를 명시적으로 세우고 각 목표를 하나씩 풀어가는 방식**을 보여준다. 대량 생산이 가능한 새로운 물리적 구조를 고안·검증하고, 재생지의 단면 노출처럼 예상치 못한 디테일 문제를 뒤늦게 발견해 표지 재질을 통째로 바꾸는 등, 작은 디테일까지 놓치지 않으려는 태도가 반복적으로 드러난다. 완성된 스케줄러는 이후 펀샵(온라인 편집숍)에서 정식 판매되기도 했다.

## 핵심 내용
- 2005년 개인용 종이 시간표에서 출발해 전 직원이 쓰는 사내 도구로 자연스럽게 확산
- UX 행사 후원을 계기로 외부 배포용 제품으로 재디자인 — 5가지 목표(아이덴티티 컬러·예시 페이지·달력 공존·탁상형·UX 회사다운 관점) 설정
- 직원 사용 행태 관찰과 반복 테스트로 시간 테이블 레이아웃·줄 간격 결정
- 기존 탁상형의 E자 연결 구조 한계를 내부 연결 방식과 종이 텐션으로 극복한 신규 구조 고안(특허출원)
- 인쇄소와의 반복 회의·샘플 제작으로 신규 구조의 대량 생산 가능성 검증
- p·x·d 세 글자를 상징하는 타공 디자인 — 재생지 단면 노출 문제 발견 후 흰색 표지로 전면 재제작
- 겹판 뒷받침대 내부 연결 종이에 코팅 처리로 내구성·슬라이딩감 확보
- 완성 후 펀샵을 통한 정식 외부 판매로 이어짐
