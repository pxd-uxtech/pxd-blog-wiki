# UI 패턴 원칙 도서

**UI 패턴과 원칙**을 이해하는 것은 건축가가 구조를 이해하는 것과 같이 기본이면서도 중요하다. 패턴과 원칙은 일반적인 디자인 문제에 대한 광범위한 해결 방안을 제공하며, UI 기획자와 디자이너 모두에게 공통 언어 역할을 한다.

위승용(uxdragon)이 2013년 정리한 20권 이상의 목록은 웹·모바일·정보 구조·콘텐츠 등 여러 영역을 아우른다. 국내 번역 출판 기준으로 선별된 이 도서들은 pxd 라이브러리에도 구비되어 있다.

2021년 위승용은 목록에 포함된 **Designing interfaces**(제니퍼 티드웰)의 3판 번역서 **사용자를 끌어들이는 UX/UI의 비밀**에 대한 독후감을 남겼다. 2007년 초판 이후 아이폰 등장으로 디스플레이 환경이 데스크톱·모바일·패드·워치·키오스크까지 촘촘히 확장된 변화에 맞춰, 3판은 모바일 디스플레이를 다루는 챕터가 새로 추가되고 심미적 디자인 관련 내용이 대폭 보강됐다. **위치 인식·카메라·음성 인식·제스처 입력·햅틱 피드백**처럼 데스크톱에는 없는 모바일 고유 기능을 활용한 패턴들이 최신 사례로 갱신된 점이 특징이다. 리뷰는 이 책을 "UI 패턴과 활용법을 명확히 알고 기획에 적용해야 하는" UI 디자이너의 필수 참고서로 꼽으면서도, 스크린샷의 색 재현을 위한 매끄러운 재질의 종이가 페이지 두께 감소와 맞물려 밑줄 필기나 책장 넘김이 뻑뻑해지는 점을 아쉬운 부분으로 지적했다. 번역은 pxd의 **박재현**, **한상택**이 맡았다.

## 핵심 내용

**웹·일반 UI 패턴**
- **웹 폼 디자인** (루크 로블르스키, 인사이트, 2009) — 입력 폼의 레이아웃·레이블·에러 처리 원칙
- **리치 인터페이스 디자인** (빌 스콧·테레사 닐, 인사이트, 2010) — AJAX 시대의 인터랙티브 UI 패턴
- **소셜 인터페이스 디자인** (크리스천 그림리시·에린 멀론, 인사이트, 2011) — 소셜 네트워킹 UI 패턴
- **검색 패턴** (제프 칼렌더·피터 모빌, 한빛미디어, 2010) — 검색 UX의 12가지 패턴
- **웹사이트 해부하기** (로버트 후크만·제라드 스풀, 인사이트, 2012) — 실제 사이트 분석 기반 UI 설계 인사이트
- **Designing web navigation** (제임스 콜백, 한빛미디어, 2008) — 웹 내비게이션 구조 설계
- **웹 스타일 가이드** (사라 호튼·패트릭 린치, 인사이트, 2011) — 웹 콘텐츠 구조·타이포그래피·접근성
- **The design of sites** (더글러스 벤 듀인 외, 에이콘출판사, 2011) — 사이트 디자인의 패턴 카탈로그
**UX 원칙 종합**
- **Designing interfaces** (제니퍼 티드웰, 한빛미디어, 2007) — GUI 컴포넌트·레이아웃·내비게이션 패턴의 고전. 3판은 **사용자를 끌어들이는 UX/UI의 비밀**(박재현·한상택 번역, 2021)로 번역 출간되며 모바일 디스플레이 챕터가 추가됨
- **경험 디자인의 요소** (제시 제임스 가렛, 한솜미디어, 2006) — 전략·범위·구조·스켈레톤·표면 5단계 프레임워크
- **퍼소나로 완성하는 인터랙션 디자인 About Face 3** (앨런 쿠퍼 외, 에이콘출판사, 2010) — **목표 지향 디자인** 원칙의 정수
- **UX 디자인 7가지 비밀** (박지수·김현, 안그라픽스, 2013) — 국내 저자의 UX 원칙 정리
- **오래가는 UX 디자인** (반준철, 한빛미디어, 2013) — 시간이 지나도 유효한 UX 설계 철학
- **콘텐츠 UX 디자인** (제니스 레디쉬, 위키북스, 2011) — 사용자 맥락에 맞는 콘텐츠 구조화
- **모든 기획자와 디자이너가 알아야 할 사람에 대한 100가지 사실** (수잔 웨인쉔크, 위키북스, 2012) — 인지·행동심리학 기반 100가지 설계 원칙
- **Human Computer Interaction 개론** (김진우, 안그라픽스, 2012) — HCI 이론의 체계적 입문

**모바일 UI 패턴**
- **아이폰 앱 디자인 정석** (조시 클라크, 한빛미디어, 2012) — iOS 앱 UI 패턴과 제스처 설계
- **모바일 디자인 전략** (조셉 카트먼·리차드 팅, 위키북스, 2011) — 모바일 맥락 기반 UX 전략
- **모바일 애플리케이션 UI 라이브러리** (노주환, 멘토르출판사, 2011) — 모바일 UI 컴포넌트 참고서
- **UI디자인을 잘 하기 위한 테크닉 80** (노주환, 비비컴, 2002) — 절판, 초기 UI 패턴 국내 저작
