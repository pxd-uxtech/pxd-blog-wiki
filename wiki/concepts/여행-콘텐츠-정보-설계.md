# 여행 콘텐츠 정보 설계

**여행 콘텐츠 정보 설계**는 낯선 여행지의 복잡한 정보를 독자가 쉽게 이해하고 활용할 수 있도록 구조화하는 작업이다. 시중의 일반 가이드북은 정보량이 많고 내용이 딱딱해 여행 초보자들이 접근하기 어려운 문제가 있다. 이는 독자가 지명·음식·현지 용어에 생소한 상태에서 가이드북을 접하기 때문이다. GUI 디자이너 안진언은 '인사이드 플래닛'이라는 여행 가이드북 프로젝트에서 이 문제를 **캐릭터 서사**와 **일러스트레이션** 중심의 편집 방식으로 해결하고자 했다.

## 핵심 개념

- **가이드 캐릭터**: 독자가 감정이입할 수 있는 캐릭터를 통해 여행 정보를 전달. 인사이드 플래닛의 '구름이' 캐릭터는 홍콩을 처음 알아가고 여행 계획을 세우는 과정을 독자와 함께 경험한다.
- **일러스트레이션 기반 편집**: 텍스트 중심 정보를 만화책·그림책처럼 시각적으로 재구성해 정보 접근 장벽을 낮춘다.
- **콘텐츠 구조화(IA)**: 여행의 흐름에 맞춰 챕터를 설계. 홍콩 이해 → 여행 준비 → 상황별 플랜 → 스팟 가이드 → 마카오 → 맵북 부록 순서로 독자의 여행 준비 과정을 따라간다.
- **직접 취재 기반 콘텐츠**: 자체 취재한 지역 정보를 지도(맵북)로 시각화해 실용성을 높인다.

## 실무 적용

여행 콘텐츠뿐 아니라 복잡한 도메인의 정보를 일반 사용자에게 전달할 때 적용 가능한 원칙이다. **초심자 관점에서 정보 진입 장벽 파악** → **서사 구조(스토리라인) 활용** → **시각 언어(일러스트·아이콘·맵) 적극 활용**의 순서로 접근한다. 전문 용어를 먼저 설명하거나 챕터 순서를 사용자의 실제 경험 흐름에 맞추는 것이 핵심이다. 디지털 서비스에서는 온보딩 플로우, 도움말 콘텐츠, 앱 내 가이드 설계에 동일한 원칙이 적용된다.
