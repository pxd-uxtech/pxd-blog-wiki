# 컴퓨터 케이스 커스터마이징

천편일률적인 **컴퓨터 케이스**에서 벗어나 개성을 표현하려는 사용자들의 시도는 하드웨어 성능은 그대로 유지하면서 외형만 바꿔 **전체적 가치**를 변화시키는 활동이다. 클래식 기타, 대형 송풍기, 트랜스포머, 바이크, 트럭 엔진, 모아이 석상, 해골, 변기, 헐크 피규어, 바베큐 그릴, 심지어 마운틴 듀로 수냉식 냉각을 구현한 사례까지 극단적 다양성이 존재한다.

이는 단순한 취미를 넘어 **GUI 작업**과 본질적으로 같은 구조를 공유한다. 내부 기능(하드웨어 성능)을 유지하면서 겉을 바꿔 사용자 경험을 전혀 다르게 만드는 것은, 동일한 데이터와 기능 위에서 GUI 스킨을 갈아입혀 브랜드·감성·친밀도를 재설계하는 작업과 닮아있다. 제품의 **기능**과 **표현**을 분리해 바라보는 관점은 산업 디자인과 디지털 인터페이스 디자인을 관통하는 공통 주제이다.

## 핵심 내용
- 컴퓨터 케이스 개조: 성능 유지 + 외형 변화 = 가치 변화
- **기능과 표현의 분리**는 산업 디자인과 GUI 작업에 공통된 접근 방식
- 개성 표현의 극단적 사례들: 기타·바이크·석상·헐크 피규어·변기 등
- 표면을 바꿔 전혀 다른 경험을 만드는 **모드(mod) 문화**
